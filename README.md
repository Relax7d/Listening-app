# 英语听力朗读应用 - 使用说明

## 🎯 功能特性

- ✅ 英文文本朗读（支持美音/英音）
- ✅ AI随机生成英文文本
- ✅ 支持多个API提供商（DeepSeek、智谱AI、百度、阿里）
- ✅ 可调节语速
- ✅ 播放控制（开始/暂停/停止）
- ✅ 多主题生成（故事/新闻/对话/学习材料/短文）

## 📖 使用说明

### 1. 配置API
- 点击"配置"按钮
- 选择API提供商
- 输入API Key
- 保存配置

### 2. 生成文本
- 选择主题（故事/新闻/对话/学习材料/短文/随机）

### 3. 朗读文本
- 选择发音（美音/英音）
- 调整语速（0.5x - 2x）
- 点击"▶️ 开始朗读"

### 4. 播放控制
- ⏸️ 暂停/继续
- ⏹️ 停止
- 🗑️ 清空文本

## 🆘 常见问题

### Q: 点击生成没有反应？
A: 检查是否启动了后端服务：`node server.js`

### Q: 提示"请先配置API Key"？
A: 点击"配置"按钮，输入您的API Key

### Q: 生成失败？
A: 检查网络连接和API Key是否正确

### Q: 没有声音？
A: 检查浏览器是否支持语音合成，尝试刷新页面
